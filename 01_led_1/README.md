# 1LED - LED parpadeando con SysTick

Proyecto mínimo que hace parpadear un LED en **PB12** usando **SysTick no bloqueante**.  
Sin HAL, sin delay bloqueante. C puro sobre registros.

---

## 🔌 Conexión

| Componente | Pin |
|------------|-----|
| LED | PB12 |

# 1LED - Мигание светодиодом с SysTick

Минимальный проект, который мигает светодиодом на **PB12** с использованием **SysTick** (неблокирующий таймер).  
Без HAL, без блокирующего delay. Чистый C поверх регистров.
