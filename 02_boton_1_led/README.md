# boton1led - Un botón controla un LED

Proyecto que enciende un LED (PA5) mientras se mantiene pulsado un botón (PA0).  
Sin HAL, sin interrupciones. C puro sobre registros.

---

## 🔌 Conexión

| Componente | Pin |
|------------|-----|
| Botón (pulsador) | PA0 |
| LED | PA5 |
